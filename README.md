## 🚀 Getting Started

### Prerequisites

You only need a modern web browser — no installs, no Node.js, no build tools.

### Run Locally

**1. Clone the repository**

```bash
git clone https://github.com/Darshilking208/developer-portfolio.git
```

**2. Navigate into the project folder**

```bash
cd developer-portfolio
```

**3. Open in your browser**

```bash
# Simply open index.html in any browser
open index.html       # macOS
start index.html      # Windows
xdg-open index.html   # Linux
```

> 💡 For the best experience, use **Live Server** in VS Code (`Right-click index.html → Open with Live Server`).

## 🎨 Customization Guide

Making this portfolio your own is straightforward — no framework knowledge needed.

### ✏️ Update Personal Info

Open `index.html` and look for these sections:

```html
<!-- Hero Section — update name, tagline, and CTA links -->
<h1>Hi, I'm <span>Darshil</span></h1>
<p>IT Student | Web Developer | Open Source Enthusiast</p>

<!-- About Section — edit your bio paragraph -->
<p>Your bio text here...</p>
```

### 🖼️ Replace Images

Drop your images into the `assets/images/` folder and update the `src` attributes in `index.html`:

```html
<img src="assets/images/profile.jpg" alt="Darshil K. Prajapati" />
```

### 🎨 Change Colors / Theme

Open `style.css` and update the CSS custom properties at the top:

```css
:root {
  --accent-color: #6E40C9;    /* Primary purple accent */
  --bg-dark: #0d1117;         /* Dark mode background */
  --bg-light: #f9fafb;        /* Light mode background */
  --text-primary: #ffffff;
}
```

### 📦 Add a New Project Card

In `index.html`, copy an existing project card block and update the title, description, and links:

```html
<div class="project-card">
  <h3>Project Name</h3>
  <p>Short description of what the project does.</p>
  <a href="https://github.com/Darshilking208/project">GitHub</a>
  <a href="https://your-live-demo.com">Live Demo</a>
</div>
```
